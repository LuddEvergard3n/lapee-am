# a11y.js

Controles de acessibilidade: preferências visuais, som e atalhos de teclado.

## API pública

```js
import {
  applyAll,
  toggleFonteGrande, toggleAltoContraste, toggleTemaEscuro, toggleSom,
  registerKeyboardShortcuts,
  focusById,
} from './a11y.js';
```

### `applyAll()`

Aplica todas as preferências do store ao DOM (`<html>`):
- `fonteGrande` → classe `.fonte-grande`
- `altoContraste` → classe `.alto-contraste`
- `temaEscuro` → classe `.tema-escuro`
- `somAtivado` → `setMuted(!prefs.somAtivado)` em `sound.js`

Chamado na inicialização e a cada `setState({ prefs })` via subscriber.

### `toggle*()`

Cada função lê o estado atual, inverte o campo correspondente, chama `setState({ prefs })` e `applyAll()`.

### `registerKeyboardShortcuts()`

Atalhos globais via `document.addEventListener('keydown')`:

| Tecla | Ação |
|---|---|
| Alt + F | toggleFonteGrande |
| Alt + C | toggleAltoContraste |
| Alt + D | toggleTemaEscuro |
| Esc | fecha sidebar/drawer |

### `focusById(id)`

Move foco para elemento com tabindex=-1. Disponível para uso pós-navegação em leitores de tela.

## Exposição global

`window._a11y = a11y` em `app.js` permite acesso em `ui/paginas.js` sem criar dependência circular.
